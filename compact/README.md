# @agentdev/compact

compact feature for AgentDev.

## Installation

```bash
npm install @agentdev/compact
```

## Usage

```typescript
import { Agent } from 'agentdev';
import { Compact } from '@agentdev/compact';

const agent = new Agent().use(new Compact());
```

## Development

```bash
npm install
npm run build    # 或 npm run dev 监听模式
```
